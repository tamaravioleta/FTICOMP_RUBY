GARBAGE COLLECTION

The garbage collection is responsible for freeing up the memory that is done
automatically as long as the java program is still active. In addition, programmers are also freed
of the load to reallocate the memory)

JAVA VIRTUAL MACHINE

JVM or Java Virtual Machine is a virtual machine that works like a real machine.
The JVM provides a hardware specification that can compile all java programs.
